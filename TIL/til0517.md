미들웨어
추가기능  app.use / 모든 미들웨어는 함수이다. 

```js
// 미들웨어 = 함수
function helloMiddleware(req, res, next) {
  console.log('hello')
  next()
}

app.use(helloMiddleware)
```
미들웨어는 함수이고 안에서 무슨 작업이던지 가능하다.
헤더를 주입하고 정보를 빼내서 작업을 하고 등등 모든 작업들을 할 수 있다.
